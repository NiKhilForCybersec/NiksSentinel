Common SOAR Use Cases

1. Automated Enrichment

Trigger: Any alert with IP artifact

Playbook Steps:
1. Extract IP from container
2. Query VirusTotal for reputation
3. Query AbuseIPDB for reports
4. Query internal asset database
5. Query Shodan for open ports
6. Add enrichment data to container
7. Adjust severity based on findings

2. Phishing Response

Trigger: Phishing email reported

Playbook Steps:
1. Parse email headers and body
2. Extract URLs, attachments, sender
3. Check URLs against threat intel
4. Detonate attachments in sandbox
5. Search mailboxes for similar emails
6. If malicious:
   a. Delete emails from all mailboxes
   b. Block sender domain
   c. Block malicious URLs at proxy
7. Notify reporter of outcome
8. Create case for tracking

3. Malware Containment

Trigger: Malware detection from EDR

Playbook Steps:
1. Get host details from EDR
2. Check file hash against threat intel
3. Get process tree
4. If high confidence malware:
   a. Isolate host (network quarantine)
   b. Kill malicious processes
   c. Collect forensic artifacts
5. Create ServiceNow ticket
6. Notify SOC team via Slack
7. Update case with findings

4. Account Compromise

Trigger: Suspicious login activity

Playbook Steps:
1. Get user details from AD
2. Get recent authentication logs
3. Check for impossible travel
4. Get user's recent file access
5. If compromise indicators:
   a. Force password reset
   b. Revoke active sessions
   c. Enable MFA if not active
6. Notify user's manager
7. Escalate to Tier 2
