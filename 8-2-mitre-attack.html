ATT&CK Navigator

CREATING A COVERAGE LAYER:
1. Open ATT&CK Navigator
2. Create new layer
3. For each detection:
   - Select technique
   - Set score (0-100 for coverage level)
   - Add comment with detection name
4. Export layer as JSON
5. Track changes over time

SCORE MEANINGS:
0 = No detection
1 = Partial detection (signature only)
2 = Good detection (behavior-based)
3 = Excellent detection (multiple methods)

"How do you use MITRE ATT&CK in your detection engineering process?"

1. Threat Research: Use ATT&CK to understand attack techniques, data sources needed, and real-world procedures.
2. Detection Development: Map each detection to specific tactics and techniques, ensuring the detection addresses the documented behavior.
3. Coverage Assessment: Use ATT&CK Navigator to visualize what techniques we can detect, identify gaps, and prioritize improvements.
4. Threat Intelligence: Map threat actor TTPs to our coverage to understand exposure to specific threats.
5. Communication: Use ATT&CK as a common language with stakeholders, red team, and management to discuss security posture.
