"How do you validate that your detections are working?"

1. Attack Simulation: Use Atomic Red Team to execute controlled tests mapped to MITRE ATT&CK techniques. Run tests in isolated environment and verify alerts trigger.
2. Historical Validation: Run detection queries against historical data to check for expected matches and unexpected false positives.
3. Continuous Monitoring: Track detection metrics (true positive rate, false positive rate, alert volume) over time.
4. Purple Team Exercises: Coordinate with red team to test detection coverage during exercises.
5. Regular Review: Quarterly review of detection effectiveness, update rules based on findings.

Chapter Summary

  • Test detections through unit, integration, FP, and performance testing
  • Atomic Red Team provides portable attack simulations
